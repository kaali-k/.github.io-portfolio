KSD Protector: Secure Password Manager

A secure, offline-first mobile application for generating, analyzing, and storing your passwords.

Download APK (GitHub Release)

Technology Stack

  • Core Framework: Built with React Native, allowing for a cross-platform codebase that runs natively on Android.
  • Local Storage: Utilizes `@react-native-async-storage/async-storage` for persistent, on-device data storage, ensuring all credentials and the security PIN are stored locally and are accessible offline.
  • Security & Encryption: Implements robust AES encryption using `crypto-js` for the export/import functionality, protecting the user's data vault with a user-provided passphrase.
  • Navigation: Features a clean, state-based navigation system for a fast and lightweight two-screen user flow (Generator & Vault).
  • File & Data Management (Export/Import): Leverages `react-native-fs` to save encrypted backups to the device's Downloads folder and `react-native-document-picker` to allow users to securely restore their vault from a backup file.
  • Native API Integration:
    • `PermissionsAndroid` for robust, runtime permission handling for storage access.
    • `Linking` to direct users to app settings if permissions are permanently denied.
    • Native Clipboard API for easy one-tap copying of passwords.
    • `BackHandler` for custom back-press behavior, including a "press back again to exit" feature.
  • User Interface & Feedback: Provides user feedback through custom-built, non-intrusive animated "toasts" and confirmation modals for critical actions like deletion and app exit.
  • Component Library: Utilizes specialized components like `react-native-date-picker` for a native date selection experience.
  • PDF Generation: Financial reports are generated as PDF documents using `react-native-html-to-pdf`.

About This Project

KSD Protector is a security-focused tool designed for straightforward and secure management of your digital credentials. With a very simple, clean, and cyber-themed user interface, this application allows you to generate strong passwords and keep your account details all in one secure, offline place.

The core principle of this app is data privacy and user control. Unlike cloud-based password managers, all your sensitive information is stored exclusively on your device, giving you complete ownership and peace of mind.

Key Features:

  • Advanced Password Generator: The ability to generate highly secure passwords with customizable length (8-64 characters) and character sets (uppercase, lowercase, numbers, symbols).
  • Password Security Analyzer: A built-in tool to analyze the strength of any password, providing a real-time strength meter ("Vulnerable" to "Fortress") and a detailed security checklist.
  • Secure Credential Vault: Easily record and manage all your account credentials, including service name, username/email, password, and optional notes.
  • Encrypted Data Backup & Restore: The facility to Export your entire vault into a single, passphrase-protected encrypted file (.cvb) and Import it to restore data. (Note: This feature is currently non-functional and requires further development.)
  • PIN Protection: The entire vault is secured with a user-configurable PIN, which is required every time you access the vault section.
  • One-Tap Copying: Securely copy your passwords to the clipboard with a single tap for easy logins.
  • Offline-First Design: The application is fully functional without an internet connection, ensuring your data is always accessible to you.

Development Focus:

This project demonstrates the ability to build a security-conscious mobile application that prioritizes user privacy. Development focused on a simple yet effective architecture, ensuring reliability and data integrity. Significant emphasis was placed on creating a highly responsive and intuitive user interface within its dark, cyber-themed aesthetic. The permission system was carefully designed to be user-friendly, guiding the user to the device settings when necessary. The system is designed for complete offline operation, providing users with absolute control over their sensitive credential data.
