K.S.D Personal Finance Tracker & Manager

A powerful and intuitive mobile application designed to simplify your daily financial management.

Download APK (GitHub Release)

Technology Stack

  • Core Framework: Built with React Native, allowing for a cross-platform codebase that runs natively on both Android and iOS.
  • Local Database: Utilizes SQLite via `react-native-sqlite-storage` for persistent, on-device data storage, ensuring that all financial information is stored securely and is accessible offline.
  • Navigation: Implemented using React Navigation (`@react-navigation/native-stack`) to create a smooth and intuitive multi-screen user flow.
  • Theming System: A dynamic theming system was built using React's Context API, allowing users to seamlessly switch between Light and Dark modes.
  • Data Management (Export/Import): Uses the native Share API for exporting data and a custom modal for pasting and importing data, providing a reliable backup solution without dependency issues.
  • File Handling (Legacy): Initially integrated `react-native-fs` and `react-native-document-picker` for direct file system access.
  • User Interface & Experience (Gestures): `react-native-gesture-handler` is used for advanced touch interactions like the swipe-to-delete/edit functionality in lists.
  • Notifications: User feedback is provided through non-intrusive "toasts" using `react-native-toast-message`.
  • Component Library: Utilizes specialized components like `react-native-date-picker` for a native date selection experience.
  • PDF Generation: Financial reports are generated as PDF documents using `react-native-html-to-pdf`.

About This Project

"MyFinance App" is an innovative solution designed for the easy and systematic management of your personal finances. Featuring a very simple, clean, and user-friendly interface, this application allows you to keep your income, expenses, and reminders all in one place.

Key Features:

  • Income & Expense Tracking: The ability to easily record all your income and expenses with categories and notes.
  • Financial Reports: The ability to generate detailed reports including income, expenses, and net balance for any date range you choose.
  • Reminders & Notes: The ability to add reminders with dates for important payments, bills, or anything else.
  • Data Backup & Restore: The facility to Export and Import your valuable financial data, keeping it safe even if you change your phone.
  • Dark Mode Support (Customizable Themes): The ability to switch between Light and Dark themes according to your preference.
  • Secure User Accounts: A personal account system secured with a password.

Development Focus:

This project demonstrates the ability to build a full-featured, user-friendly mobile application from concept to deployment, handling everything from local database management to native API integration. Development prioritized a modular and maintainable architecture, ensuring scalability and data integrity. Significant emphasis was placed on creating a highly responsive and user-friendly interface, ensuring a smooth and efficient user experience across various devices. The system is designed for easy local deployment, providing users with complete control over their financial data and operations.
